# Sport turborepo starter

This is an sport site official starter Turborepo.

# Using this project

Run the following command:

```
pnpm i
```

Dev mode

```
turbo dev
```

Build production

```
turbo build
```

# Our stack

-   Turborepo
-   Turbopack
-   NextJS ^v13.4
-   TypeScript
-   pnpm
-   Zustand
-   MUI
-   SCSS
-   Storybook
